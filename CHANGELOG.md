Supported archives
===

- [x] TAR `"tar"`

Supported compressions
===

- [x] BZIP2
- [x] GZIP
- [x] DEFLATE
- [x] LZ4 (BLOCK, FRAMED)
- [x] LZMA
- [x] XZ
